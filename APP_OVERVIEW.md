# バフェットAIスクリーナー (Buffett AI Screener) - 機能概要

このプロジェクトは、ウォーレン・バフェットの投資哲学に基づき、AI（Gemini）を活用して株式のスコアリングおよび診断を行うウェブアプリケーションです。

## 🚀 主な機能

### 1. AI株式診断
- ティッカーシンボル（例: AAPL, 7203.T）を入力することで、企業の財務データを自動取得し、AIがウォーレン・バフェットの視点で分析を行います。
- **総合スコア**: 0〜100点満点で企業の投資魅力を数値化。
- **一言要約**: 企業の現状をバフェットらしい表現で簡潔に解説。

### 2. 多角的なメトリクス分析
- **定量的データ**: ROE、自己資本比率、PER、利益率などの重要指標をリアルタイム（Yahoo Finance）で取得・表示。
- **定性的分析**: ブランド力（モート）、経営陣の質、競争優位性、リスク要因をAIが評価。

### 3. ユーザーインターフェース
- **モダンなデザイン**: Next.jsとTailwind CSSを使用したレスポンシブでプレミアムなUI。
- **ダークモード対応**: `next-themes`によるシステム連動・手動切り替え。
- **診断履歴**: 過去に分析した銘柄を履歴として保存（今後の拡張予定）。

---

## 🛠 技術スタック

- **フレームワーク**: Next.js (App Router), TypeScript
- **スタイリング**: Tailwind CSS (Vanilla CSS & モダンユーティリティ)
- **AIエンジン**: Google Gemini API (分析・評価ロジック)
- **データベース**: Supabase (分析結果のキャッシュ、永続化)
- **データソース**: Yahoo Finance API (株式データ取得)
- **インフラ/開発支援**: Stitch & MCP Server

## 🧩 開発環境・連携

本プロジェクトは、効率的な開発と高度な機能拡張のために以下のツールを連携させています。

### Stitch (スティッチ) の活用
- **UI/UXデザイン**: デザインシステムの一貫性を保ちながら、洗練されたインターフェースを迅速に構築しました。
- **プロジェクト管理**: UIコンポーネントの生成からコードの更新まで、一元化されたワークフローで開発を進めています。

### MCP (Model Context Protocol) サーバー設定
- **高度な連携**: MCPを介して、外部データソースやツール、ファイルシステムへのシームレスな対話を可能にしています。
- **基盤設定**: 開発エージェントがプロジェクトの構造を正確に理解し、コンテキストに沿った修正を即座に行えるよう、MCPサーバーとの接続を確立・最適化しました。これにより、AIによる自律的な機能追加やバグ修正の精度が向上しています。
