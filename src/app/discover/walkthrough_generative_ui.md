# ハイブリッド型銘柄検索・評価フロー (Generative UI) 実装レポート

## 概要
Vercel AI SDK の `streamUI` を活用し、自然言語での銘柄探索からバフェット流の詳細評価までをシームレスに行える「対話型ハイブリッドUI」を実装しました。

## 実装された主要機能

### 1. AI 探索ページ (`/discover`)
- 会話形式で銘柄を検索できるチャットインターフェース。
- セッション内でのメッセージ履歴管理（リロードでリセット）。
- 自動スクロールおよびローディングアニメーション。

### 2. インテリジェント・ツール
- **`search_stocks`**: ユーザーの曖昧な検索意図（例：「割安な半導体銘柄」など）をベクトル変換し、Supabaseのベクトル検索 (`match_tickers`) を用いて類似銘柄を抽出します。成果物はチャット内に「銘柄カード」として直接レンダリングされます。
- **`analyze_stock`**: Yahoo Finance からリアルタイムの財務データ（ROE, PER, 自己資本比率など）を取得し、バフェットの投資哲学に基づいた詳細評価レポートを生成します。

### 3. ハイブリッド・インタラクション
- AIが提示した「銘柄カード」のボタンをクリックすると、その銘柄に対する詳細評価リクエストが自動的にチャットに送信され、流れるように分析が開始される連携機能を実装しました。

### 4. 堅牢なインフラ
- **Gemini モデルフォールバック**: `gemini-2.5-flash` → `gemini-2.5-flash-lite` → `gemma-4-31b-it` の順でレート制限を自動回避するロジックを Server Action に組み込みました。
- **高性能ベクトル検索**: `transformers.js` (`all-MiniLM-L6-v2`) をサーバーサイドで実行し、クライアントに負荷をかけずに高度な意味検索を実現しています。

## 画面構成
- **ボトムナビゲーション**: 「AI探索」タブを追加。アイコンには `explore` を採用し、直感的なアクセスを提供。
- **レスポンシブ・メッセージ**: ユーザーとAIで異なるバブルスタイルを適用。銘柄カードはグリッドレイアウトで美しく整列されます。
